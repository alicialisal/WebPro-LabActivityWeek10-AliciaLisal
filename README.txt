##Cara menggunakan Formulir Registrasi

Formulir Registrasi ini dapat merecord nama Panjang, tanggal lahir, email, nomor telepon, gender, program of interest, alamat, nama ibu, dan nama ayah, dan foto.

Beberapa aturan pengisian formnya adalah:
1. Kolom Full Name (nama Panjang), Mother's Name (nama ibu), dan Father's Name (nam ayah) tidak dapat diisi dengan huruf ataupun symbol. User hanya dapat mengetikkan huruf ke dalamnya, baik huruf besar (kapital), huruf kecil, dan spasi.
2. Kolom nomor telepon hanya dapat diisi dengan angka dan symbol "-". Nomor telepon harus diisi sesuai format yang disediakan. Untuk saat ini, nomor telepon hanya dapat diisi untuk format Indonesia, yaitu 4 angka diawal (termasuk 0), lalu pisahkan dengan "-", lalu lanjut lagi dengan 4 angka, dan pisahkan lagi dengan "-", lalu lanjutkan dengan 3 - 5 digit terakhir.
3. Kolom tanggal lahir (date of birth) tidak dapat diisi jika tanggal lahir kurang dari batas umur 17 tahun sebelumnya. Jika tanggal kurang, maka isi date of birth akan tereset.
4. Kolom email akan dianggap tidak valid jika format tidak sesuai dengan format yang diterima. Misalnya, jika tidak mengandung "@" dan ".". Selain itu, posisi titik minimal dibelakang 2 huruf terakhir dan posisi "@" tidak boleh berada di paling awal dan di paling akhir teks email.
5. Gender dapat dipilih sesuai gender user masing-masing dan wajib untuk diisi.
6. Program of Interest juga dapat diisi sesuai keinginan user masing-masing dan wajib untuk diisi.
7. Gambar dapat dipilih dari direktori local masing-masing user dan wajib untuk diisi.

Lalu, jika semua kolom sudah selesai diisi sesuai aturan di atas, maka user dapat memencet tombol Submit. Selanjutnya, data yang diisi di formular akan tampil di bawah formulir dan user dapat membacanya. Jika sudah sesuai, user dapat memencet tombol Confirm dan Input New Form. Lalu akan muncul teks confirm sebagai pertanyaan konfirmasi terakhir, jika sudah benar-benar yakin, anda cukup mengklik Oke agar formulir dapat di-reset dan user dapat menginput form baru.
